#  Blood Donation Network

A web-based platform that connects blood donors, recipients, hospitals, and blood donation camps to make blood donation and blood requests faster, more organized, and more accessible.


#  Project Overview

The **Blood Donation Network** is a web application developed to simplify the process of blood donation and emergency blood requests. The platform enables donors to register, recipients to search for suitable donors, hospitals to manage blood requests, and users to stay informed about upcoming blood donation camps.

This project aims to promote voluntary blood donation while supporting **United Nations Sustainable Development Goal (SDG) 3 – Good Health and Well-being**.


# Objectives

- Develop a centralized blood donation management platform.
- Connect blood donors, recipients, hospitals, and blood donation camps.
- Enable users to search for blood donors based on blood group and location.
- Simplify blood request and donation processes during emergencies.
- Promote awareness about voluntary blood donation.
- Improve accessibility to blood donation services.
- Support SDG 3 – Good Health and Well-being.


# Modules

##  1. Home Module
Acts as the landing page of the application.

### Features
- Website introduction
- Blood donation awareness banner
- Latest announcements and campaigns
- Emergency contact numbers
- Navigation menu


##  2. About Blood Donation Module

Educates users about blood donation and spreads awareness.

### Features
- Benefits of blood donation
- Donor eligibility criteria
- Frequently Asked Questions (FAQs)
- Blood donation awareness
- Educational resources


##  3. User Management Module

Manages user accounts and authentication.

### Features
- User Registration
- User Login
- User Profile Management
- Secure authentication
- Store donor information


##  4. Donor Search Module

Allows recipients to search for suitable blood donors.

### Features
- Search donors by blood group
- Search donors by city, district, or area
- View donor availability
- Display donor contact information


##  5. Blood Request Module

Allows recipients or hospitals to request blood during emergencies.

### Features
- Submit blood requests
- Enter patient details
- Select blood group
- Request blood units
- Track request status
- Emergency request support


##  6. Blood Donation Camp Module

Provides information about blood donation camps.

### Features
- View upcoming camps
- Online camp registration
- Camp schedule
- Eligibility information
- Organizer contact details


##  7. Contact & Support Module

Provides communication between users and administrators.

### Features
- Contact form
- Email support
- Phone support
- Technical assistance
- User query management

##  8. Reviews & Feedback Module

Collects user feedback for continuous improvement.

### Features
- Submit ratings
- Write reviews
- Display testimonials
- Feedback management
